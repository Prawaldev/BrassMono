# Brass Mono Font for Revenge Manager

A working Brass Mono font package compatible with **Revenge Manager (Discord Mod)**.

---

## 📌 Overview
This repository provides a ready-to-use configuration of the Brass Mono font that can be imported directly into Revenge Manager using its built-in font import system.

---

## 🚀 Installation Guide

### 1. Open Revenge Manager
Install and open **Revenge Manager**. After setup, you will see several customization options.

---

### 2. Navigate to Fonts
- Go to the **Fonts** section inside Revenge Manager
- Click on the **"+" (plus) icon**
- Select **"Import font from entries via link"**

---

### 3. Get the Import Link

You need a direct JSON raw link:

- Open the JSON file in this repository
- Click on **"View Raw"**
  - Or right-click and select **"Copy link address"** (Desktop)
  - On mobile, long-press the **View Raw** button and copy the link

---

### 📎 Raw JSON Link
https://raw.githubusercontent.com/6bpr/working-font/refs/heads/main/json/theme.json

---

### 4. Paste the Link
- Paste the copied raw link into Revenge Manager’s import field
- Confirm import
- The Brass Mono font will now be available in your fonts list

---

## ✅ Done
You have successfully installed the Brass Mono font into Revenge Manager.

---

## 🧾 Notes
- Make sure the link is a **raw GitHub URL**
- Do not modify the JSON file unless you know what you're doing
